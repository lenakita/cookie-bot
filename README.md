# Cookie Bot
## Setup
To setup the bot run the commands below.
```Bash
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -r requirements.txt

# generate keys file
touch keys.txt
# add in the format below. These are available from the discord API portal
# token:<token>
# id:<id>

# to run the bot
python3 discord_bot.py
```
